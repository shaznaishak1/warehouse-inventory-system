 Warehouse Inventory Management System

 Description
This is a simple Java-based inventory management system that allows users to manage items in a warehouse.

 Features
- Add new items
- Remove items
- Update item quantity
- Search items by ID
- Search items by name
- Display all items

 Technologies Used
- Java
- HashMap (Data Structure)
- OOP Concepts

  Project Structure
- Item.java → Defines item properties
- Inventory.java → Handles inventory operations
- Warehouse.java → Main program with menu system

How to Run
1. Compile all files:
   javac *.java

2. Run the program:
   java Warehouse
